Storybook-Stories
=================

- Stories under `stories/` (Views, components); only for Storybook / visual tests.
- Not part of the production client bundle.
- Configuration: `../main.ts`, `../preview.ts`.
- Storybook Mocks (composable/module stubs): `../mocks/` (e.g. `useDashboardState`).
- Shared Story Data (static example/test data): `fixtures/` (e.g. `matchStoryHelpers`, `rosterStoryHelpers`).
